1、定义和语法

A source location or source URL locator refers to an OVF package, VMX file, or virtual machine in a VMware product, such as VMware Workstation, vSphere ESXi Host, vSphere vCenter Server, vCloud Director, or vFabric Data Director.

A target location or destination URL locator specifies either a file location, or a location within a VMware product, such as VMware Workstation, vSphere ESXi Host, vSphere vCenter Server, vCloud Director or vFabric Data Director

基本语法如下：

ovftool <source locatior> <target locator>

ovftool <options> <source locatior> <target locator>

ovftool -h

和是虚拟机，OVF package，OVA package或vSphere location的源和目标的路径

2、命令行选项

Table 2-1. OVF Tool Definitions of Source and Target Locators

Locator Definition
Path to the source, which must be either a virtual machine, vApp, vApprun workspace entity, or an OVF package。
The source locator can be one of the following:
A path to an OVF or OVA file (a local file path, or an HTTP, HTTPS, or FTP URL).
A virtual machine (a local file path to a .vmx file).
A vSphere locator identifying a virtual machine or vApp on vCenter, ESXi, or VMware Server.
A vCloud Director locator identifying a virtual machine or a vApp in vCloud Director.
A local file path to a vApprun workspace entity
The target locator can be one of the following:
A local file path for VMX, OVF, OVA, or vApprun workspace.
A vSphere locator identifying a cluster, host, or a vSphere location.
A vCloud Director locator identifying a virtual machine or a vApp in vCloud Director.

3、示例

ovftool -o --powerOffSource --acceptAllEulas --machineOutput --noSSLVerify --noImageFiles <ovf-dir> file.ova

virt-v2v -i ova file.ova -o local -os <out_dir> -of raw --quiet

从 vSphere 导出虚机ova

ovftool -o --powerOffSource --acceptAllEulas --machineOutput --noSSLVerify --noImageFiles vi://root@Password:<vSphere-IP>:443/<vm-name> ./file.ova
