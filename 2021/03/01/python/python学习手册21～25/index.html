24、模块包

包要素

到Python 3.3版本为止，包导入语句中的路径中的每个目录内都必须有 __init__.py文件。在3.3之后的版本，没有这一文件的模块路径会被当作单路径命名空间包导入，其工作方式相同，但是没有初始化代码的执行。

包相对导入

命名空间包

Python 3.3 版本（以及之后的版本）现在拥有两种风格的包：

  • 原始的模型，现在称为常规包
  • 可选的模型，称为命名空间包
